# Claude Skills

This directory contains `.skill` files used during AI Days sessions. Skills are portable packages of instructions and tools that extend what Claude can do.

## How to Install a Skill

1. **Download the `.skill` file** — Click on the file in this repository, then click the download button (or "Raw" and save the file).
2. **Open Claude** — Go to [claude.ai](https://claude.ai) or open Claude Desktop.
3. **Import the skill** — In a Claude conversation, click the attachment icon and select "Add Skill", then choose the `.skill` file you downloaded.
4. **Use the skill** — The skill will be available in your conversation. You can invoke it by name or by describing what you want to do.
