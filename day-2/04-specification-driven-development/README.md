# Specification-Driven Development: Building Research Tools Without Coding

**Thursday, March 5, 2026**  
**1:40 PM – 3:10 PM** (90 minutes)

**Presenter:** Myranda Shirk

## Session Description

Drawn from Course 4: Agentic Solution Development and Deployment. Topics: Specification writing as primary development artifact, directing agentic coding tools, test-driven development with agents, evaluation of agent output. Faculty learn to write clear specifications and use AI coding agents to build functional tools — data processing scripts, visualization dashboards, analysis pipelines — without prior programming experience. The emphasis on specification writing over coding is what makes this accessible to non-CS faculty.
