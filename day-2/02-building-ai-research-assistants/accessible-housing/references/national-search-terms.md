# Nationally Relevant Program Search Terms

Use these terms when expanding research to new states. Each state will have its own names for these program types. Search for the **state-specific equivalent** of each.

## Federal/State Waiver & Program Types

| Search Term | What It Is |
|---|---|
| 1915(c) HCBS Waiver | Home and Community-Based Services waivers — every state has at least one; many have several for different populations (I/DD, aging, physical disability, TBI, etc.) |
| 1915(i) State Plan HCBS | HCBS services available to Medicaid beneficiaries without a waiver slot |
| 1915(k) Community First Choice | Optional Medicaid state plan benefit for community-based attendant services |
| 1115 Demonstration Waiver | Experimental Medicaid programs — some states use these for managed LTSS or housing supports |
| Money Follows the Person (MFP) | Federal program helping people transition from institutions to community living |
| PACE | Program of All-Inclusive Care for the Elderly — community-based managed care for nursing-home-eligible individuals |
| ICF/IID | Intermediate Care Facilities for Individuals with Intellectual Disabilities — state-operated or private institutions |
| Olmstead Plan | Each state's plan for transitioning people from institutions to integrated community settings |

## Housing-Specific Federal Programs

| Search Term | What It Is |
|---|---|
| Section 8 / Housing Choice Voucher | Tenant-based rental assistance — administered by local Public Housing Authorities |
| Section 811 | HUD Supportive Housing for Persons with Disabilities |
| Section 202 | HUD Supportive Housing for the Elderly |
| LIHTC | Low-Income Housing Tax Credit — state Housing Finance Agency allocates; some set-asides for disability |
| HOME Investment Partnerships | Federal block grant for affordable housing |
| National Housing Trust Fund | Funding for extremely low-income housing |
| Continuum of Care (CoC) | HUD homeless assistance including permanent supportive housing |
| Project-Based Vouchers | Section 8 vouchers attached to specific properties |
| Emergency Solutions Grants | HUD funding for homelessness prevention and rapid re-housing |

## State Agency Types to Search

| Search Term | What To Find |
|---|---|
| State DD Agency | The department managing intellectual/developmental disability services (e.g., DIDD in TN, DDS in CA, OPWDD in NY) |
| State Mental Health Authority | Department managing mental health services and supportive housing (e.g., TDMHSAS in TN) |
| State Housing Finance Agency | Administers LIHTC, Section 811 PRA, HOME funds (e.g., THDA in TN) |
| Public Housing Authority | Local agency managing Section 8 and public housing (e.g., MDHA in Nashville) |
| Protection & Advocacy | Federally mandated legal advocacy org in every state (e.g., Disability Rights TN) |
| UCEDD | University Center for Excellence in Developmental Disabilities — one in every state |
| ADRC | Aging and Disability Resource Center — single point of entry for long-term services |
| Area Agency on Aging | Regional agencies coordinating services for older adults |

| Best Buddies International | Inclusive independent living programs in select cities |
| Easterseals | Local affiliates may operate residential programs |
| L'Arche | Intentional community homes in select locations |
| Goodwill Industries | Some affiliates offer supportive housing |
| United Cerebral Palsy | Local affiliates may operate housing or navigation |
| NAMI | Local chapters may connect to supportive housing |
| Autism Society | Local chapters may have housing initiatives |
| Jewish Family Services | Some affiliates provide supported housing |
| Catholic Charities | Some affiliates provide housing programs |
| Volunteers of America | Operates housing in many states |
| Salvation Army | Permanent supportive housing in some cities |
